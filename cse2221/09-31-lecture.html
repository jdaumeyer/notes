1. Pointers and References

With Primitive Types in Java, the value is directly assigned the variable.

Reference Values operate in a very different manner. Instead of their variable being directly assigned a value, they are instead assigned a pointer to an address in memory.

1.1. Assignment

When assigning reference types to copy another, it is improper form to use an = assignment as this will actually point both variables to te same object. 

NaturalNumber n = new NaturalNumber(3);
NaturalNumber n2 = n;

Both n and n2 in the code above will be equal to the same object. Meaning if the value of n is changed, so it n2. The proper way to copy objects would be with a copy constructor. For Natural Number this would look like: 

NaturalNumber n3 = new NaturalNumber(n);
