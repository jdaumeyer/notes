Mathematical String Notation

Lecture Slides A mathematical string is a series of zero or more entries of any other mathematical type. This automatically makes it a bit different then the Java String type which is only a series of char types. This type is going to be used in the examples for recursion below.

Notation

The notation for a mathematical string is by having a series of elements between \(<,>\).

  • \(< 1, 2, 3, 4>\)
  • \(< `G`, `o`, `o`, `g`, `l`, `e`>\)
  • \(< 1, 2, `a`, `b`>\)

Concatenation

Mathematical string concatenation is represented by a \(*\) symbol as oppossed to the Java + symbol for concatenation.

Length

Mathematical string length is denoted by \(| |\). So \(|< 1, 2, 3, 4>|\) would be 4

Substrings

A mathematical string \(s\) would be a substring of \(t\) if the entries of \(s\) appear consecutively in \(t\)

  • \(s\) is a prefix if the \(s\) appears as a substring of \(t\) at the start of \(t\)
  • \(s\) is a prefix if the \(s\) appears as a substring of \(t\) at the end of \(t\)

Recursion

Lecture Slides Recursion is a very important and influential topic in computer science. At its most fundamental level, recursion is simply when a function calls itself. This is done by breaking up a larger problem into a smaller problem.

#+header :cmdline -cp ../components.jar 

public static int power(int n, int p) {
    if (p == 0) {
        return 1;
    }
    return n * power(n, p - 1);
}

public static void main(String[] args) {
    SimpleWriter out = new SimpleWriter1L();
    out.print(power(5, 5));
    out.close();
}

Base Case

All recursive functions need one key element, a base case. This is a situation where the method returns a value without calling itself. This is what prevents a recursive function from executing infinitely.

  • When a function does execute infinitely, Java will return a StackOverflowError.

Examples

Recursive Factorial

Program a function that recursively calculates the factorial \(n!\). This would be \(n \times n-1 \times n-2 \times n-\ldots\) 

private static int factorial(int n) {
    // Base Case
    if (n == 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

Fibonacci Sequence

The Fibonacci Sequence is a common mathematical sequence most known for its use in the Golden Ratio. The sequence is defined as \(F_n = F_{n-1} + F_{n-2}\) where \(F_0 = 0\) and \(F_1 = 1\). Program a recursive function that calculates the \(n^{th}\) Fibonacci Number. 

private static int fibonacci(int n) {
    // Base Case
    if (n == 1 || n == 0) {
        return n;
    } else {
        return fibonacci(n - 1) + fibonacci(n - 2);
    }
}

You have a tool that can cut logs that are length 1, 2, and 4. Given a log of length \(L\), how many possible times could you cut the log? 

private static int cutLog(int l) {
    sum = 0

    if (l > 1) {
        sum += cutlog(l - 1);
    }
    if (l > 2) {
        sum += cutLog(l - 2);
