Object Oriented Programming

Abstract Classes

An Abstract Class is what bridges the gap between an Interface and a Class. They can’t have objects instantiated from them and are allowed to have abstract methods.

Kernel Class

Kernel Classes aren’t really a thing outside of Ohio State. But a Kernel Class is essentially just a class that has a data representation.

  • They have a data representation that can be manipulated
  • There are rules on how the data represents the object value
  • There are rules on how the data is stored

We want to keep the Kernel class as small as possible and implement as much as we can in the Abstract class.

  • Because everything depends on the Kernel class, they can be hard to debug.

Template Method

Template Method is a design pattern for dealing with different implementations of the same thing.

The Object Class

Object is a super class of every concrete type. The object class defines three methods that all objects will have:

  • toString()
  • equals()
  • hasCode()
