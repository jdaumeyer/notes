What is a Limit?

Limits as a Physics Problem

Limits were actually created by Physicists to solve the problem of finding out the speed of something at a certain time. The problem being that if we use our equation for speed at a certain time, we get a division by zero. So limits are used to look near the point but not at the point.

Notation

\(\lim\limits_{x \to 1} f(x) = 1\) the \(x \to 1\) shows the variable and value we’re looking at. \(f(x)\) is the function we are looking at and the \(= 1\) is the value the function is approaching.

One-Sided Limits

A function may actually behave differently on different sides of a point, so we need a way to analyze each side separately.

Left-Sided Limits
or limits that are approaching a value slightly less than \(a\) are represented by \(\lim\limits_{x \to a^-} f(x)\)
Right-Sided Limits
or limits that are approaching a value slightly more than \(a\) are represented by \(\lim\limits_{x \to a^+} f(x)\)

One-Sided Limits implicitly prove the existence of Two-Sided Limits. These are actually what a normal Limit is, Two-Sided Limits only exist if both One-Sided Limits exist and they are equal to each other.

Continuity

Graphs that are Continuous are simple and easy to use. However, graphs that have Discontinuity can be tricky to deal with. We need to develop a language to deal with Continuity.

The definition of continuity is \(\lim\limits_{x \to a} f(a)\). If this equation is true, then \(f(x)\) is continuous at \(a\). This definition is really making three statements:

  • \(f(a)\) is defined, That is, \(a\) is in the domain of \(f\)
  • \(\lim\limits_{x \to a} f(x)\) exits
  • \(\lim\limits_{x \to a} f(x) = f(a)\)

One-Sided Continuity

Since our definition of Continuity is built on Limits and limits have sides, this means that there is also One-Sided Continuity.

A function \(f\) is Left Continuous at a point \(a\) if \(\lim\limits_{x \to a^-} f(x) = f(a)\) and a function \(f\) is Right Continuous at a point \(a\) if \(\lim\limits_{x \to x^+} f(x) =f(x)\).

  • If a function is Continuous at \(x=a\), then it is also Left-Continuous and Right-Continuous.
  • If a function is Left-Continuous and Right-Continuous at \(x=a\), then is is also Continuous at \(x=a\)

Intervals of Continuity

Our previous definitions of Continuity have been based on points. However, we can also judge is a function is Continuous on an interval.

\(f(x)\) is continuous over the interval \((a, b)\) if it is continuous at every point in the interval.

  • When looking at Intervals of Continuity, it is very important to check endpoints.

Limit Laws

We really care about Limit Laws because our current definition of Limits is very vague and makes it hard to actually evauluate limits. So Limit Laws are the first tool in our toolbox.

The Laws, for these examples assume \(\lim\limits_{x \to a} f(x) = L\) and \(\lim\limits_{x \to a} g(x) = M\) and both \(L\) and \(M\) exist. If any limit doesn’t exit when using a limit law, then it can’t be solved with limit laws.

Law 1
All Famous Functions are continuous over their domain.
Law 2
The Sum Law, if we have two functions begin added together, we can break them into two smaller limits. \(\lim\limits_{x \to a} (f(x) \pm g(x)) = [ \lim\limits_{x \to a} f(x) ] \pm [ \lim\limits_{x \to a} g(x)] = L * M\)
Law 3
The Product Law. \(\lim\limits_{x \to a} (f(x) * g(x)) = [\lim\limits_{x \to a} f(x)] * [\lim\limits_{x \to a} g(x)] = L * M\)
Law 4
The Quotient Law, \(\lim\limits_{x \to a} \frac{f(x)}{g(x)} = \frac{ [\lim\limits_{x \to a } f(x)] } {[\lim\limits_{x \to a} g(x)]} = \frac{L}{M}\). However the Quotient Law only works if \(M \neq 0\)
Law 5
The Composition Law, \(\lim\limits_{x \to a} [f(g(x)] = f( [ \lim\limits_{x \to a } g(x)]) = f(M)\)

Limit Laws and Continuity

  • If \(f\) and \(g\) are continuous on an interval \(I\), then \(f \pm g\) is also continuous on \(I\)
  • If \(f\) and \(g\) are continuous on an interval \(I\), then \(f * g\) is also continuous on \(I\)
  • If \(f\) and \(g\) are continuous on an interval \(I\), then \(\frac{f}{g}\) is also on continuous on \(I\) only if \(g(x) \neq 0\)
  • If \(f\) and \(g\) are continuous on \(x = a\), then \(f(g(x))\) on is not continuous at \(x = a\). This would only work if \(f(x)\) is continuous at \(x = g(a)\).

Squeeze Theorem

Take for example \(\lim\limits_{x \to 0} x * sin(1/x)\). This problem looks like it can be solved with our Limit Laws however to use our laws we need both limits to exits and \(\lim\limits_{x \to o} sin(1/x)\) does not exist. This means that we can’t use the Product Law to get the \(x\) separated.

To solve for a limit like this, we can use the Squeeze Theorem.

  • The Squeeze Theorem is based on the fact that if both \(f(x)\) and \(g(x)\) are defined over an interval \((a, b)\) and \(c\) is a real number where \(a < c < b\). And if \(f(x) \leq g(x)\) for all \(x\) in \((a, b)\) and both \(\lim\limits_{x \to c} f(x)\) and \(\lim\limits_{x \to c} g(x)\) both exit, then \([\lim\limits_{x \to c} f(x)] \leq [\lim\limits_{x \to c} g(x)]\)

The Squeeze Theorem uses this fact to suppose that \(g(x) \leq f(x) \leq h(x)\) for all \(x\) near \(a\) but not necessarily \(x = a\). Then if \(\lim\limits_{x \to a} g(x) = L = \lim\limits_{x \to a} h(x)\), then \(\lim\limits_{x \to a} f(x) = L\).

When using the Squeeze Theroum on test, be sure to note that you are using the Squeeze Theorem!

What are forms?

Forms are what you get as a result of plugging in the number of a limit. There are a lot of forms but the ones we’re focusing on right now are:

Non-zero over non-zero
\(\frac{\#}{\#}\)
Zero over non-zero
\(\frac{0}{\#}\)
Non-zero over zero
\(\frac{\#}{0}\)
Zero over zero
\(\frac{0}{0}\)

Forms are used to show us the method we should use to solve for a limit.

Key Notation

When describing the form of a limit, do not say \(\lim\limits_{x \to 0} \frac{sin(x)}{x} = \frac{0}{0}\), instead say “\(\lim\limits_{x \to 0} \frac{sin(x)}{x}\) is in the form of \(\frac{0}{0}\)”.
