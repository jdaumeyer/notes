The derivative functions are very useful for analyzing graphs. Each derivative can tell us something different about a function.

Increasing / Decreasing

A function is increasing on the interval \(I\) if \(f(x_1) < f(x_2)\) for all points in \([x_1, x_2]\) where \(x_1 < x_2\). And \(f\) is decreasing on an interval \(I\) if \(f(x_1) > f(x_2)\) for \(x_1 < x_2\).

  • Essentially if the value of \(f\) is getting larger than \(f\) is increasing but if it is getting smaller than it is decreasing.
  • To find if a function is increasing or decreasing without a graph, we can use a derivative. The function \(f\) is increasing where \(f^\prime(x) > 0\) and decreasing where \(f^\prime(x) < 0\).

Concavity

Concavity is the general trend of a function to go up or down. There are two types of concavity, Concave Up and Concave Down.

_20211018_183140image002.png

To determine the concavity of a function on an interval, we use the Second Order Derivative of the function ( the derivative of the derivative ).

  • \(f\) is concave up on an interval \(I\) if \(f^{\prime\prime}(x) > 0\).
  • \(f\) is concave down on an interval \(I\) if \(f^{\prime\prime}(x) < 0\).

Sign Charts

To look at the concavity of an entire functions, it is very helpful to make a sign chart. This is done by finding the Critical Points of a function. Critical Points are places where a function exists and \(f^{\prime}(x) = 0\) or \(f^\prime(x)\) does not exist. Then we determine the values of \(f^\prime(x)\) and \(f^{\prime\prime}(x)\) between these points.

Inflection Points

An Inflection Point is a point where a function goes from being concave up to being concave down or where \(f^{\prime\prime}(x) = 0\).

Implicit and Explicit Functions

There are actually two different types of functions in mathematics, however up until this point you have probably only dealt with Explicit Functions. These are the types of functions that are defined explicitly in terms of \(x\) (or some other variable).

However there is another type of function. These are Implicit Functions, or graphs that are defined in both terms of \(x\) (or any other depended variable) and \(y\) (or any other dependent variable).

Implicit Derivatives

Since Implicit Functions can be graphed, they can have tangent lines. This means that we can take the derivative of an Implicit Function. However we must use a different method in order to do so, this method is Implicit Differentiation.

Linear Approximation

The formula for a tangent line can be found as \(y= f^\prime(a) (x - a) + f(x)\). This is the key equation for Linear Approximation where we are trying to find an estimate of an equation at a point by looking at points close to it. Effectively we do this by finding the point on the equation’s tangent line. This means that \(L(x) = f^\prime(a) (x - a) + f(a)\) where \(a\) is our base point and \(x\) is the point we are trying to find.

The Language of Differentials

Differentials are basically just a more formal version of Linear Approximation, but where with Linear Approximation we are trying to estimate the value of the function Differentials find how much the function is changing. Now despite being so similar to a Linear Approximation, Differentials have a strict syntax and method.

  1. Find a function (\(f(x)\), \(g(x)\), \(h(z)\), etc).
  2. Determine your base input (often \(a\) but sometimes \(x\)).
  3. The base point gets moved a distance of \(dx\) or \(\Delta x\).
  4. We want to estimate \(\Delta f\) which is equal to \(f(x) - f(a)\).
  5. We can use Linear Approximation to estimate \(f(a)\).
  6. Use the estimate to solve for \(\Delta f\).
