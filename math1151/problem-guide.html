1. Limits

The Laws, for these examples assume \(\lim\limits_{x \to a} f(x) = L\) and \(\lim\limits_{x \to a} g(x) = M\) and both \(L\) and \(M\) exist. If any limit doesn’t exit when using a limit law, then it can’t be solved with limit laws.

Law 1
All Famous Functions are continuous over their domain.
Law 2
The Sum Law, if we have two functions begin added together, we can break them into two smaller limits. \(\lim\limits_{x \to a} (f(x) \pm g(x)) = [ \lim\limits_{x \to a} f(x) ] \pm [ \lim\limits_{x \to a} g(x)] = L * M\)
Law 3
The Product Law. \(\lim\limits_{x \to a} (f(x) * g(x)) = [\lim\limits_{x \to a} f(x)] * [\lim\limits_{x \to a} g(x)] = L * M\)
Law 4
The Quotient Law, \(\lim\limits_{x \to a} \frac{f(x)}{g(x)} = \frac{ [\lim\limits_{x \to a } f(x)] } {[\lim\limits_{x \to a} g(x)]} = \frac{L}{M}\). However the Quotient Law only works if \(M \neq 0\)
Law 5
The Composition Law, \(\lim\limits_{x \to a} [f(g(x)] = f( [ \lim\limits_{x \to a } g(x)]) = f(M)\)

3.2. First Fundamental Theorem

if \(f(x)\) is continuous then,

\begin{align*} \frac{d}{dx} [\int_a^x f(t) \hspace{0.25em} dt] = f(x) \end{align*}

3.3. Second Fundamental Theorem

if \(F'(x) = f(x)\) then,

\begin{align*} \int_a^b f(x)\hspace{0.25em} dx = F(b) = F(a) \end{align*}

3.4. Riemann Sums

Riemann Sums are a method of determining the area under a curve. The steps to set one up are:

  1. Know the function and interval you’re finding the area of.
  2. Split the interval into \(n\) sub-intervals of equal length, to find the Grid Points
  3. Use Grid Points to calculate the Base Points, using the formula for left, right or mid end points.
  4. Use the Base Points to find the height of each rectangle by plugging them into to function.
  5. Sum up the area of each rectangle by multiplying the height (the value of the function) and width (the \(\Delta x\)).
    1. \(\sum_{k=1}^{n} c = c \cdot n\)
    2. \(\sum_{k = 1}^{n} k = \frac{n(n + 1)}{2}\)
    3. \(\sum_{k=1}^{n} k^2 = \frac{n (n + 1) (2 n + 1)}{6}\)

4.4. Linear Approximation

The formula for a tangent line can be written as \(y = f'(x)(x - a) + f(a)\).

This is also used for Differentiables. The process for this is:

  1. Determine your function
  2. Find a base input (often \(a\) but sometimes \(x\))
  3. The base point gets moved a distance of \(dx\) or \(\Delta x\).
  4. We want to find \(\Delta f = f(x) - f(a)\).
  5. We use Linear Approximation to estimate \(\Delta x\).

4.5. Extreme Value Theorem / Global Extrema

The Extreme Value Theorem states that if a function \(f(x)\) is continuous over a closed interval \([a, b]\), then \(f(x)\) has a global max and min in \([a, b]\).

Using this we can find the Global Extrema of a function by the following process:

  1. Find all critical points of \(f(x)\) in \([a, b]\).
  2. Plug all critical points and end points into \(f(x)\).
  3. The largest value is the global max and the smallest the global min.

4.6. Mean Value Theorem

The Mean Value Theorem state that if \(f(x)\) is continuous on \([a, b]\) and differentiable on \((a, b)\), then there is a point \(c\) where \(a < c < b\) where \(f'(c) = \frac{f(b) - f(a)}{b - a}\).

We can use it to determine if a point exists between \(a\) and \(b\).
